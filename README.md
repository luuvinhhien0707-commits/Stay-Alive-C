# Stay-Alive-C
A console-based survival game written in C featuring wave progression, random events, and basic resource management (hunger and thirst).
# C Survival Game

A simple console survival game written in C.

## Gameplay
- Player has two stats: Hunger and Thirst
- Each wave reduces these stats
- The player must choose actions to survive
- Special events happen every 5 and 10 waves
- Random chance mechanics affect the outcome

## Features
- Wave-based gameplay
- Random events
- Basic resource management
- Replay system

## Language
C (Console Application)
